COMBAT GAME APPLICATION
Author : Joshua Pocsidio 18267495

File(s) Required to Compile:
- build.xml
- resources/item_database.txt

To Compile:
- change directory to 'combat-game' directory
- type 'ant' to clean and build the program
- dist folder should appear

To Run:
- change directory to 'dist' directory
- type 'java -jar CombatGame.jar'
